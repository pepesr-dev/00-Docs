# Composer 
**Gestor de dependencias** para php.
1. **Librerías** Descarga herramientas como ***SMARTY***.
2. **Versiones**.
3. Genera el vendor/**autoloader** para encontrar automáticamente todas las clases.

# Instalación
Necesita instalar php y LAMP.

1. Descargar el instalador.
2. Ejecutar el instalador.
3. Hacerlo global.
4. Borrar el instalador y comprobar que funcione.



# Configuración del proyecto

1. Iniciar composer en la carpeta de mi proyecto.
2. Configurar el composer.json.
3. Crear src/Coche.php que contiene la primera clase.
4. Registrar el namespace(autoloader).
5. Crear el entry point/frontcontroler (index.php).
