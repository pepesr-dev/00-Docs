# Composer - Cheat Sheets
| Elemento | Acción | Sintaxis |
| :--- | :--- | :--- |
| **namespace**| Para identifica un autor de otro | `namespace Google\servicios;` |
| **use**| Usar namespaces | `use Google\servicios;` |
| **static**| Identifican funciones propios de la clase, las que no necesitan objetos creados para funcionar | `public static function countCars(){}` |
| **protected** | Atributo que solo la clase y sus hijas puden ver | `protected ?int $id = null;` |
